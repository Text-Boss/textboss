What's in the shell

  • Header — hamburger (opens drawer), current section name, context action (+ new chat)
  • Thread strip — horizontal scrollable chips below header (Chat section only)
  • Drawer — slides in from left, shows tier badge, nav items, sign out
  • Content area — full height between header and input, section-specific
  • Input bar — pinned to bottom with iOS safe-area inset
  • Dim overlay — tap anywhere outside drawer to close it
